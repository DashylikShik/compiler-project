# MiniCompiler Project

Учебный проект по созданию компилятора (Спринт 1 - Лексический анализатор)

## Описание

Этот проект реализует первую часть компилятора - лексический анализатор (сканер).
Он преобразует исходный код на упрощенном C-подобном языке в последовательность токенов.

## Структура проекта
compiler-project/
├── src/ # Исходный код компилятора
│ ├── lexer/ # Лексический анализатор
│ │ ├── scanner.py # Сканер (читает код -> токены)
│ │ └── token.py # Определения типов токенов
│ └── utils/ # Вспомогательные модули
├── tests/ # Тесты
│ ├── lexer/ # Тесты для лексера
│ │ ├── valid/ # 20 тестов с правильным кодом
│ │ └── invalid/ # 10 тестов с ошибочным кодом
│ ├── test_scanner.py # Простой тест
│ └── test_runner.py # Автоматический запуск всех тестов
├── examples/ # Примеры программ
│ ├── test1.src # Первый пример
│ └── test2.src # Второй пример
└── docs/ # Документация
└── language_spec.md # Спецификация языка


## Установка и запуск

1. Запуск тестов:
Создание тестов (один раз)
python tests/test_runner.py --create

2. запуск тестов:
python tests/test_runner.py

3. запуск конкретного примера:
python tests/test_scanner.py
